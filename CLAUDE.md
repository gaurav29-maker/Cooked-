## Website Structure

### Current Sections
1. **Navigation** - Logo + "by Bina Rajan" + menu links
2. **Hero** - "Gourmet Vegetarian Meals, Reimagined."
3. **Mission** - About Bina Rajan
4. **Values** - Daily Fresh, Light on Oil, Effortless Convenience
5. **Menu** - Soups, Salads, Starters, Mains, Sweets (with expandable "View More")
6. **Testimonials** - Slider with 3 reviews
7. **Delivery** - Hours, Areas, Order Notice
8. **Contact** - WhatsApp, Email, Location
9. **Footer** - Logo, Instagram, WhatsApp icons

### Key Features Implemented
- Smooth scroll animations (fade-in, fade-in-left, fade-in-right, scale-in)
- Staggered menu item animations
- Mobile-responsive design
- Expandable menu sections (Starters: 30 items, Mains: 32 items)
- Sticky WhatsApp button
- Back to top button
